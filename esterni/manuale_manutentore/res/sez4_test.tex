\subsection{Unit test}
	Sono stati utilizzati diversi \glock{framework} di test. Le tre parti che compongono il progetto (Server, unità e UI) sono state codificate con linguaggi di programmazione diversi e di conseguenza anche i test:
	\begin{itemize}
		\item per il server, sviluppato in \glock{Java}, è stato utilizzato il \glock{framework} \glock{JUnit};
		\item per le unità, codificate in \glock{TypeScript}, sono state usate le suite \glock{Mocha} e \glock{Chai};
		\item per l'interfaccia, prodotta con \glock{Angular}, è stato usato il \glock{framework} \glock{Jasmine}.
	\end{itemize}
	
\subsection{Analisi statica}
	Per l'analisi statica del codice sono state utilizzate le seguenti tecnologie:
	\begin{itemize}
		\item \glock{Checkstyle} per il codice \glock{Java} del server;
		\item \glock{ESLint} per il codice della UI;
		\item \glock{Sonarcloud} per la \glock{continuous integration}.
	\end{itemize}
